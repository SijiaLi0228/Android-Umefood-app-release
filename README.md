# 🍱 UmeFood App (Android)

UmeFood 是一款为用户提供外卖点餐与美食推荐服务的移动应用。  
本仓库用于存放 **Android 发布版本 (Release APKs)**。

---

## 📦 下载与安装

在 [Releases](https://github.com/SijiaLi0228/Android-Umefood-app-release/releases) 页面中下载最新的 `app-release.apk` 文件。

安装方式：
1. 下载最新的 APK 文件；
2. 将文件传输到手机；
3. 打开并安装（如有提示，请允许“安装来自未知来源的应用”）。

---

## 🧩 功能特性

- 🍔 在线浏览ume超市商品 
- 🛒 购物车下单与支付流程
